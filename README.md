# Research Paper Analysis & Professional Booklet Generation

## 📌 Overview
This project is an advanced Retrieval-Augmented Generation (RAG) application built with Streamlit. It allows users to upload academic research papers (PDFs), ask context-specific questions, and automatically generate professional, fully-formatted PDF booklets. The application integrates sophisticated data visualizations, automated citation searching, and large language model (LLM) analysis to synthesize complex academic information into digestible reports.

## ✨ Key Features
* **Intelligent Document Processing:** Extracts and chunks text from uploaded PDFs using `pypdf` with configurable chunk sizes and overlaps.
* **Local Vector Database:** Utilizes ChromaDB to store document embeddings generated by the `sentence-transformers/all-MiniLM-L6-v2` model for highly relevant context retrieval.
* **Advanced AI Analysis:** Powered by Google's `gemini-2.0-flash` model, the system generates comprehensive summaries, methodology breakdowns, and answers to user queries.
* **Automated Citation Search:** Integrates with the Crossref API to automatically discover and link related research papers based on the analyzed content.
* **Professional Visualizations:** Uses Matplotlib and Scikit-Learn (TF-IDF) to dynamically generate:
    * Methodology Flowcharts
    * Comprehensive Results Dashboards
    * Research Timelines
    * Hierarchical Concept Maps
* **PDF Booklet Generation:** Compiles the AI-generated analysis, custom visualizations, and citations into a clean, professional A4 PDF using ReportLab.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **LLM Provider:** Google Generative AI (Gemini)
* **Embeddings & Database:** Sentence-Transformers, ChromaDB
* **PDF Handling & Export:** PyPDF, ReportLab
* **Data Visualization & NLP:** Matplotlib, Seaborn, NumPy, Scikit-learn (via TF-IDF)
* **External APIs:** Crossref API

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd <repository-folder>
   ```
2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Environment Variables:**
Create a .env file in the root directory and add your Google Gemini API key:
   ```bash
   GEMINI_API_KEY=your_google_gemini_api_key_here
   ```
##  Usage

   * Start the Streamlit application:
   ```bash
   streamlit run main.py
   ```
  * Open the provided local URL in your web browser.

  * Upload and Index: Upload a research paper (PDF) via the sidebar/main interface and click "Index PDF" to chunk and store the text in the local ChromaDB.

  * Ask Questions: Use the text area to query specific details about the methodology, results, or findings.

  * Generate Booklet: Click "Generate Professional Booklet" to run the full analysis, generate the Matplotlib visualizations, fetch Crossref citations, and download the final PDF report.

## Project Structure

    main.py: The core Streamlit application interface and state management.

    config.py: Centralized configuration for models, database paths, and API endpoints.

    database_manager.py: Handles ChromaDB initialization and collection management.

    pdf_processing.py: Manages the extraction and logical chunking of PDF text.

    analysis_generator.py: Constructs prompts and communicates with the Gemini API to format analytical responses.

    visualizations.py: Contains the AdvancedVisualizationGenerator class for parsing text and rendering analytical charts.

    citation_search.py: Connects to the Crossref API to retrieve external research metadata.

    booklet_generator.py: Manages document styling, layout, and PDF compilation using ReportLab.

    utils.py: Helper functions for displaying visuals seamlessly within Streamlit.
